# NJUPT-SAST-Python-WoC-2025

The repository for 2025's NJUPT SAST Python Winter of Code.

Click [here](https://njupt-sast.feishu.cn/wiki/GS5Vw6IDNiieVIkcf9MczzHInVf) to check out question details and submission requirements.

## Submission Directions

First, fork this repository, develop in a branch, and submit a PR to the corresponding branch of this repository upon completion.

> Due to the difficulty of the questions, you can submit PRs during development and ask questions, marking them as "in progress". Consideration will be given during the review.
